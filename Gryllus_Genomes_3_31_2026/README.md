# Genome File For Gryllus pennsylvanicus and Gryllus firmus

These files are going to be apart of the upcoming manuscript `The complete genomes of two hybridizing field crickets, Gryllus pennsylvanicus and Gryllus firmus` (Grapin et. al. In Prep) 

**Please do not share without asking permission from Dr. Erica Larson `erica.larson@du.edu` first.**

To check integrity of files upond recieving run ```md5sum -c checksum.md5``` in this directory. Contact mgrapin2@unl.edu if any of the files fails.
